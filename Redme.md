A professional windows app exe for GaussianHaircut hair reconstruction workflow, featuring geometry-aware splatter control and FBX export curves suitable for Maya workflows. It can automatically generate 3D Gaussian splatters from a single image, outputting Maya-compatible FBX curves with high fidelity, referencing GaussianHaircut.

UI: Input box (left): Use Custome-Tkinter as base UI framework,,Allows uploading images or videos (compatible with multiple formats). Output box (left): Interactive 3D view previewing the generated Gaussian splatters and final curves; a button to switch the final output display mode (GaussianSplatting/Curves); a download button for direct download, with selectable download format: MayaCurve FBX or Blender glb.
